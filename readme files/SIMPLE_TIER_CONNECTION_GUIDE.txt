================================================================================
                    PART 4: HOW TO DRAW IN DRAW.IO
================================================================================

DRAWING A DATA TIER BOX:
------------------------------------------------------------------------------

1. Select Rectangle tool

2. Draw rectangle (300px Ã— 80px)

3. Format:
   - Fill color: Light shade (blue, purple, orange, or green)
   - Border: 3px
   - Border color: Darker shade

4. Add title:
   - Double-click box
   - Type name (e.g., "File Storage")
   - Format: Bold, 16pt, centered


DRAWING A CONNECTION ARROW:
------------------------------------------------------------------------------

1. Click Connector tool

2. Click bottom edge of Logic Tier module

3. Click top edge of Data Tier box

4. Format arrow:
   - Width: 3pt
   - Color: Blue (#2196F3)
   - End Arrow: Classic, Size 7
   - Pattern: Solid

5. For bidirectional arrows:
   - Also set Start Arrow: Classic, Size 7

6. For return flows (dashed):
   - Set Pattern: Dashed

7. Add label:
   - Double-click arrow
   - Type label text
   - Format: Bold, 11pt


MAKING ARROWS LOOK CLEAN:
------------------------------------------------------------------------------

1. Keep arrows vertical (straight down from Logic to Data)

2. Avoid crossing arrows:
   - Group related connections together
   - Use slight curves if needed

3. Align connection points:
   - Arrows should come from bottom center of Logic modules
   - Arrows should go to top center of Data boxes

4. Consistent spacing:
   - Keep parallel arrows evenly spaced


================================================================================
                    PART 5: ARROW TYPES EXPLAINED
================================================================================

DOWNWARD ARROWS (Solid, pointing down):
   - Logic Tier sends data to Data Tier
   - Examples: Storing files, saving reports, logging events
   - Color: Blue
   - Style: Solid

UPWARD ARROWS (Dashed, pointing up):
   - Data Tier returns data to Logic Tier
   - Examples: Loading files, loading models
   - Color: Blue
   - Style: Dashed

BIDIRECTIONAL ARROWS (Arrows on both ends):
   - Data flows both ways
   - Examples: Cache operations, session management
   - Color: Blue
   - Style: Solid
   - Both ends have arrows
