                    PART 1: UNDERSTANDING THE ML PIPELINE
================================================================================

THE COMPLETE ML PIPELINE HAS 5 MAIN STAGES:

1. INPUT STAGE - Raw ECG signal enters
2. PREPROCESSING STAGE - Signal is cleaned and normalized
3. MODEL STAGE - U-Net processes the signal
4. POST-PROCESSING STAGE - Extract medical information
5. OUTPUT STAGE - Diagnosis and explanations


THE U-NET ARCHITECTURE HAS 3 PARTS:

1. ENCODER (Downsampling Path) - Extracts features
2. BOTTLENECK (Deepest Layer) - Attention mechanism
3. DECODER (Upsampling Path) - Reconstructs output
