COLAB TRAINING GUIDE - ATRIONNET HYBRID RESEARCH
============================================

1. PREPARATION
--------------
- Zip your "ml_component" folder into a file named: ml_component.zip
- Upload "ml_component.zip" to the main folder of your Google Drive.
- Open a new Google Colab notebook.
- Set Runtime to GPU: Menu -> Runtime -> Change runtime type -> T4 GPU.
