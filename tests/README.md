## AAMI EC57 Metric Definitions

Per **AAMI EC57 §3.1** — Beat Detection:

```
Se  (Sensitivity)         = TP / (TP + FN)
+P  (Positive Predictivity) = TP / (TP + FP)
FPR (False Positive Rate)  = FP / (TP + FP)
Jaccard (Temporal)         = TP / (TP + FP + FN)
```

**Matching criteria:** A detection is counted as TP if the predicted peak centre  
falls within `±50ms` (tight) or `±100ms` (loose) of the ground-truth P-wave interval.  
At 500 Hz: ±50ms = ±25 samples; ±100ms = ±50 samples.

**AAMI minimum thresholds:** Se ≥ 0.75, +P ≥ 0.75.

## Reproducibility Guarantee

The test split is guaranteed identical to training by replaying:
```python
np.random.seed(42)
perm     = np.random.permutation(N)       # same as train.py line 73
idx_test = perm[int(N * 0.85):]           # same as train.py line 80
```
This ensures **zero data contamination** between training and evaluation.
