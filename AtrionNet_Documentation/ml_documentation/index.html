ML Component Deep-Dive

The machine learning component is the core diagnostic engine of AtrionNet, specifically designed for high-precision detection and quantification of Atrioventricular (AV) Blocks in long-term ECG recordings.

Technologies Used

Python 3.10 PyTorch (Lightning-ready) NumPy & SciPy Matplotlib (Analysis)

Core Architecture: Attentional-AtrionNet v5.0

The system utilizes a Multi-Scale CNN with Dilated Contextual Bridges. This architecture solves the P-wave vanishing gradient problem typical in standard RNN/BiLSTM implementations.

Innovation Highlights:

  • Attentional Inception Blocks: Three parallel kernel branches (Small, Med, Large) to capture polymorphic wave morphologies.
  • Dilated CNN Bridge: Replaced the old BiLSTM with 1D Dilated Conv [1, 2, 4, 8] for 20x faster training and 10x better stability.
  • Triple Head Decoder: Simultaneous prediction of Segmentation Mask, Heatmap (Distance Map), and Wave Width.

Data Pipeline & Preprocessing

A rigorous pipeline transforms raw signal data into clinical-grade diagnostic inputs.

1. Signal Denoising: Baseline wander removal using high-pass filtering (0.5Hz) and power-line interference rejection (50Hz Notch).
2. Augmentations: Dynamic noise injection, baseline shift, and random amplitude scaling to ensure robustness against sensor drift.
3. Normalization: Z-score normalization per-channel for input consistency across different ECG hardware.

Module & Code Reference

src/modeling/atrion_net.py

Defines the flagship v5.0 hybrid architecture. Contains the Inception modules and the Dilated Bridge logic.

src/inference/predictor.py

The production entry point. Manages the model loading, multi-channel inference, and translation of tensors back to clinical PR/RR intervals.

src/data_pipeline/instance_dataset.py

Custom PyTorch dataset implementation for ECG segments, handling sliding window extraction and label mapping.

src/engine/atrion_evaluator.py

Comprehensive benchmarking tool. Calculates F1-Score, Sensitivity, and Specificity specifically for P-wave and QRS complex overlaps.

Explainable AI (XAI) Implementation

AtrionNet utilizes Activation Heatmap Regression to let cardiologists see *why* the AI flagged a specific region as an AV-block. The model projects confidence scores onto the raw signal, highlighting discordant P-waves with high-intensity markers.
