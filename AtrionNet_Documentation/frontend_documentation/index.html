Frontend & User Experience

The AtrionNet Clinical Dashboard is a modern, high-intensity interface designed for cardiologists to review and validate complex AV-block diagnoses in record time.

UI Design Principles

Diagnostic Focus: High-contrast ECG plotting with interactive overlays for P-wave and QRS complex masks.
XAI Accessibility: Visualizing AI confidence scores directly on the signal timeline through heatmap gradients.
Responsive Grid: Single-screen clinical workflow that adapts from standard monitors to widescreen diagnostic stations.

Advanced Technology Stack

React 18+ Tailwind CSS 3.0 Vite JS Canvas/SVG Overlay Axios / TanStack Query

Page-Level Architecture

src/pages/HomePage.jsx

Entry portal for uploading ECG studies. Handles drag-and-drop validation for industrial-standard signal formats (.npy).

src/pages/AnalysisPage.jsx

The "Cockpit" of the system. Displays real-time analysis results, AI-generated heatmaps, and provides access to medical report downloads. Integrated with complex state-management to handle large-scale signal data transitions.

src/pages/InstructionsPage.jsx

Resource hub for clinicians, detailing the clinical interpretation of the AI's "severity" metrics and color-coding schemes.

Feature & Structural Components

src/components/features/ECGViewer.jsx

Proprietary component for rendering physiological signals. Implements high-performance scaling to ensure 500Hz signals remain fluid during browser interactions.

src/components/features/AnalysisMetrics.jsx

Dynamic dashboard component for heart-rate variability, PR-interval consistency, and diagnostic severity indicators.

src/services/api.js

Centralized Axios instance. Manages environment-aware API URL resolution and response interceptors for clinical-grade error handling.

Signal Visualization System

The system uses a Multi-Layer Visualization Strategy:

  • Base Layer: Raw analog signal samples (subsampled for performance).
  • Mask Layer: P-Associated, P-Dissociated, and QRS region highlights.
  • Activation Layer: AI Heatmap showing morphological relevance of various signal peaks to the final diagnosis.
